# 神經網路解說短片 — 使用說明

本說明對應專案內 **`projects/how-neural-networks-learn/`** 的 Manim 動畫與 **`production_script.json`** 劇本流程：完成劇本後可一鍵產生**畫面 + 中英字幕 + 旁白**的 MP4。

---

## 1. 進專案、開啟虛擬環境

在 WSL 或 Linux 終端機執行：

```bash
cd ~/OpenMontage/projects/how-neural-networks-learn
source .venv/bin/activate
```

若尚未建立虛擬環境，請先於此目錄執行：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install manim edge-tts
```

（系統需已安裝 **ffmpeg**；Manim 編譯依賴請參考該子目錄內既有說明或 `install_and_render.sh`。）

---

## 2. 編輯「劇本」

用編輯器開啟：

**`projects/how-neural-networks-learn/production_script.json`**

- **`narration_zh` / `narration_en`**：各段旁白文字（會依秒數對齊畫面）。
- **`caption_zh` / `caption_en`**：畫面上顯示的**中英字幕**（建議短句）。
- **`duration_sec`**：該段長度（秒），須與 `neural_network_learning.py` 內對應區塊註解一致；若你改了動畫的 `run_time` / `wait`，請同步修改此處。

---

## 3. 一鍵出片（畫面 + 中英字幕 + 旁白）

在 **`how-neural-networks-learn`** 目錄下、且已 `source .venv/bin/activate` 後執行：

```bash
chmod +x RUN_FULL_PIPELINE.sh
NARRATION_LANG=zh ./RUN_FULL_PIPELINE.sh
```

- **`NARRATION_LANG=zh`**：旁白為中文（台灣語音預設於劇本內 `voice_zh`）。
- 若改為**英文旁白**（字幕仍為中英雙語）：

```bash
NARRATION_LANG=en ./RUN_FULL_PIPELINE.sh
```

腳本會依序：**Manim 渲染** → **依劇本產生 ASS 字幕與 WAV 旁白** → **ffmpeg 燒字幕並合成音訊**。

第一次若缺少套件：

```bash
pip install edge-tts
```

字幕中文顯示較完整建議安裝字型：

```bash
sudo apt install -y fonts-noto-cjk
```

---

## 4. 影片輸出位置

預設輸出檔（路徑相對於 **`projects/how-neural-networks-learn/`**）：

**`media/videos/neural_network_learning/1080p60/NeuralNetworkLearningExplainer_bilingual.mp4`**

完整絕對路徑範例（請依你的家目錄調整）：

```text
~/OpenMontage/projects/how-neural-networks-learn/media/videos/neural_network_learning/1080p60/NeuralNetworkLearningExplainer_bilingual.mp4
```

若曾修改 `production_script.json` 內的 **`final_mp4_relative`**，則以上檔名與子路徑以該欄位為準。
